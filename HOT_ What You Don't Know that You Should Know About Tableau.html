Rounded Bar Chart

Map

2) Map

Steps:

  • 2.1) Right Click [Location String] and select Transform → Custom Split
  • 2.2) Under Separator Enter a comma ( , ) → Change Split Off from "First" to "All"
  • 2.3) Right Click to Rename Fields
    • Location String - Split 1 → Region
    • Location String - Split 2 → State
    • Location String - Split 3 → City
    • Location String - Split 4 → Delete This Split
  • 2.4) Right Click fields again to change Geographic Role
    • State → State/Province
    • Region → Create From → State
    • City → City
  • 2.5) Double Click on [State]
  • 2.6) Add [Profit] to Color
  • 2.7) At the top of the screen, select Map → Background Layers
  • 2.8) Deselect all checked boxes
  • 2.9) Right Click on "1) Rounded Bar Chart" and select Copy Formatting then Right click on the current sheet and select "Paste Formatting"
What we learned:
- Quick string Transformations
- Map formatting
- Quick Map creation 
- Fast fact-check

Map Drill Down

  • 3.2) Drag [City] over the map and place on the pop up menu Map Layers in the top left corner

  • 3.6) Select the City Tab and Place [Profit] on Color and [Sales] on Size

Heat Map

Shout Out

5) Shout Out

Steps:

  • 5.1) Double Click [Measure Names]
  • 5.2) Drag [Measure Names] to the Filters Shelf and deselect everything Except: [Sales], [Profit], [Quantity], [Discount]
  • 5.3) CTRL + Drag [Measure Names] to Text
  • 5.4) Click the Text Button on the Marks Card select alignment and align the text to CENTER and MIDDLE respectively
  • 5.5) Edit the Text and Change the [Measure Values] and [Measure Names] Fields to your liking (we suggest Increasing the font size of [Measure Values] and changing the color of [Measure Names]
  • 5.4) Right Click [Discount] on the fields shelf and select:
    • Default Properties → Aggregation → Average
    • Default Properties → Number Format → Percent → 0 decimal points
  • 5.5) At the top of the screen change the view from “Standard” to “Entire View”

Bump Chart

6) Bump Chart - Part. 1 Steps:

  • 6.1) Right Click Drag [Order Date] to Columns and select “Year Order Date”
  • 6.2) Place [Sales] on to the Marks Card
  • 6.3) Place [Region] on to the Marks Card
  • 6.3) Double click the white space on Rows and type: Index()
  • 6.4) Materialize this Calculation by using CTRL + Drag to place it on the Fields Shelf
  • 6.5) Right Click the [Index] Field on Rows and Select “Edit Table Calculation”
  • 6.6) In the Table Calculation Editor under “Compute Using” select Specific Dimension
  • 6.7) Make sure the boxes next to “Year of Order Date” and “Region” are both Selected
  • 6.8) Make the specified selections for the following menu options:
    • At the Level → Deepest
    • Restarting every → Year of Order Date
    • Sort Order → Custom Sort → [Sales] → SUM → Desc -
  • 6.9) Add the [region] field on color

6) Bump Chart - Part. 2

  • 6.10) Edit the Axis on the left hand side by right clicking on the axis and selecting the edit option. Click the checkbox labeled "Reverse"
  • 6.11) Duplicate the [Index] function on rows by using CTRL + Drag and dragging the copy to it's right
  • 6.12) Right click the [Index] pill on the left hand side and select “Dual Axis”
  • 6.13) Right click on the left hand Axis on the view and select “Synchronize axis” from the pop up menu
  • 6.14) hide the axis by right clicking and selecting “Hide Axis”
  • 6.15) Under the second “Index” tab on the Marks Card, change the view type from “Line” to “Circle” by clicking the drop down
What we learned:
-Bump Chart
-Index function
-Dual Axis
-Materialize Calculations

Calendar

7) Calendar - Part. 1 Steps:

  • 7.1) Right Click Drag [Order Date] to Columns and select Month of Order Date
  • 7.2) Using CTRL + Drag, duplicate [MONTH(Order Date)] twice placing the new copies to the right of the original
  • 7.3) change the aggregations to YEAR and WEEKDAY respectively (weekday is located under the "more" option)
  • 7.4) Use CTRL + Drag one more time except instead of adding the copy to columns, add the new pill to rows and change the aggregation to WEEK (WEEK is located under the "more" option but label WEEK NUMBER)
  • 7.5) Add [Sales] OR [Profit] onto Color
  • 7.6) Right Click Drag [Order Date] to the filters shelf and select “Month / Year“. Filter to any Date

7) Calendar - Part. 2

  • 7.7) Create a new sheet and select the Data Source "Data Scaffold" in the top left corner
  • 7.8) Place the field [Date] on rows then right click the pill to change the aggregation:
    • Exact Date → Discrete
  • 7.9) While holding the Shift Key Select the top row and bottom row to highlight every cell in the column and use the keyboard shortcut CTRL + C (Command + C on Mac) to copy the Table
  • 7.10) Return to the Data source view for our “Sample Superstore” Data source and use the shortcut CTRL + V (Command + V) to Paste in the new data.
  • 7.11) Link the new data set to Superstore using [Order Date] = [Date]
  • 7.12) Return to the "Calendar Sheet". On the shelf, right click the field [Order Date] and select the “Replace Reference” option. when prompted replace [Order Date] with [Date]
  • 7.13) Add DAY [Date] onto text 
What we learned:
-Copy and Paste Data Set 
-Replace Reference

All Together Now! Building our Dashboard

9) Building Our Dashboard

  • 9.8) Change the toggle back to “Floating” and drag the “Map Drilldown” Sheet to the left of the map and the “Calendar Sheet” to the right of the map.
